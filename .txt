Модульные тесты

Требования к оформлению отчета по лабораторной работе
Отчет оформляется на листах формата А4 и должен содержать:
1. Титульную страницу с номером и названием лабораторной работы
фамилией студента, номером группы.
2. Условие задачи работы.
3. Описание алгоритма решения задачи в виде блок-схемы и/или
словесно.
4. Программный код реализации и снимки терминала, подтверждающие
успешное выполнение задания.
5. Выводы по работе.

Теоретический материал

Теоретический материал содержится в лекции 10 Обеспечение качества ПО.
Модульные тесты на практике».

Практическая задача

Задание 1: реализуйте класс со следующими алгоритмическими методами:
1. Поиск минимального элемента массива положительных чисел
2. Расчет суммы элементов массива, который может состоять только из
отрицательных чисел
3. Алгоритм расчета N-го элемента последовательности Фибоначчи
4. Алгоритм расчета силы тока на участке круга
Задание 2: реализуйте положительный и отрицательный модульные тесты для
каждого из четырёх методов.
Задание 3: запустите выполнение юнит тестов и подсчет покрытия кода.
Сохраните отчет по покрытию кода модульными тестами.
Задание 4: загрузите код решения на GitHub (создайте пул реквест из
добавлением MPTCourseTeacher в качестве reviwer). В отчет добавьте код
реализации, а также отчет (скриншот или таблица результатов) по покрытию
кода.
